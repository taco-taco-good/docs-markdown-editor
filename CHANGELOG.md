# Changelog

## v0.1.0-beta.1 - 2026-04-10

First public beta release.

### Highlights

- edit plain Markdown files on the web without moving them into a closed app
- keep documents as normal `.md` files inside a regular workspace folder
- use local account setup and login for a simple self-hosted start
- navigate with tabs, quick open, outline, and workspace search
- write with live preview editing for links, checklists, tables, code blocks, and dividers
- manage templates, asset uploads, and external file change detection
- deploy with Docker or a direct Node-based server run

### Beta Notes

- this release is intended for early adopters
- the recommended setup path for beta is local username/password auth
- OIDC is available and tested, but local auth is the simplest first-run path
